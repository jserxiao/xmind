# 《Go 语言圣经（The Go Programming Language）》

> 作者：Alan A.A. Donovan, Brian W. Kernighan | 难度：⭐⭐⭐进阶

## 简介

本书由 Go 语言核心团队成员和 C 语言之父联合编写，被誉为 Go 语言的"圣经"。内容深入浅出，覆盖了 Go 语言的所有核心概念。

## 目录概览

1. **教程入门**
   - Hello, World
   - 命令行参数
   - 查找重复行
   - GIF 动画
   - 获取 URL

2. **程序结构**
   - 名称
   - 声明
   - 变量
   - 赋值
   - 类型
   - 包和文件

3. **基础数据类型**
   - 整数
   - 浮点数
   - 复数
   - 布尔值
   - 字符串

4. **复合数据类型**
   - 数组和切片
   - Map
   - 结构体
   - JSON
   - 文本和 HTML 模板

5. **函数**
   - 函数声明
   - 递归
   - 多返回值
   - 错误
   - 函数值
   - 匿名函数

6. **方法**
   - 方法声明
   - 基于指针对象的方法
   - 通过嵌入结构体来组合类型
   - 方法值与表达式
   - 示例：位向量
   - 封装

## 核心亮点

- 由 Go 语言设计者亲自编写
- 大量实用示例代码
- 深入讲解 Go 的设计哲学
- 涵盖并发编程、反射等高级主题

## 学习建议

这本书适合有一定编程经验的读者。建议每章都动手敲一遍示例代码。
