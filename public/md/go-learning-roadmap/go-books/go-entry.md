# 《Go 入门指南》

> 作者：社区贡献 | 难度：⭐入门

## 简介

《Go 入门指南》是一本面向初学者的 Go 语言入门书籍，从零开始引导读者掌握 Go 语言基础。

## 目录概览

1. **Go 语言简介**
   - 为什么选择 Go
   - Go 的设计哲学
   - 环境搭建

2. **基础语法**
   - 变量与常量
   - 数据类型
   - 运算符

3. **控制结构**
   - 条件语句
   - 循环语句
   - 跳转语句

4. **函数**
   - 函数定义与调用
   - 多返回值
   - 匿名函数与闭包

5. **数组与切片**
   - 数组基础
   - 切片操作
   - 切片底层原理

6. **Map 与 结构体**
   - Map 的使用
   - 结构体定义
   - 方法与接收者

## 适合人群

- 编程新手
- 从其他语言转到 Go 的开发者
- 想要系统学习 Go 基础的读者

## 学习建议

建议配合官方文档和在线练习一起学习，理论与实践相结合。
